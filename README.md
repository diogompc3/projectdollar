# Análise da Variação Cambial (Dólar - Real)

## Sobre:
Este é um projeto de introdução à Ciência de Dados, usando para isso a linguagem Python. O script "project_dollar.py" baixa os dados dos ultimos dois anos da cotação do Dólar (USD) em relação ao Real e gera um gráfico da variação cambial.

Projeto desenvolvido durante estudos em manipulação de dados e visualização gráfica em Python.

## Linguagens e Bibliotecas utilizadas:
- Python: Linguagem escolhida
  - Pandas: Para estruturação e manipulação de dados.
  - Matplotlib: Para a construção do gráfico de linhas.
  - yfinance: Para extração das informações do API do Yahoo Finance.

## Para executar:
1. Instalar o Python
2. Instalar as dependências: 'pip install pandas matplotlib yfinance'
3. Executeo arquivo: 'py project_dollar.py'
